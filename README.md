# Система учёта личных финансов

Приложение для ведения личного бюджета: регистрация доходов и расходов, категоризация, поиск, фильтрация, визуализация в виде графиков, экспорт в CSV и PDF.

## Требования
- Python 3.10 или выше
- Библиотеки: matplotlib, reportlab
